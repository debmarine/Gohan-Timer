# **Gohan Timer**

**Gohan Timer** は、Tiger 魔法のかまどごはんの新聞紙投入タイミングを適切にお知らせするためのタイマーです。

---

## **特徴**
- ごはんの量を指定することで、最適なタイミングで通知。
- タイマー開始ボタンで簡単操作。
- Tiger 魔法のかまどごはん専用に設計。

---

## **使用方法**
1. **ごはんの量を指定**
   - 必要なごはんの量を入力します。
2. **タイマーを開始**
   - 「開始」ボタンを押すとタイマーが作動します。
3. **適切なタイミングで通知**
   - タイマーがお知らせするので、新聞紙を投入してください。
